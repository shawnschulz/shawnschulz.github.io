I’m Shawn, a current Molecular Science and Software Engineering MS student at UC Berkeley. I also got my BA in Immunology from UC Berkeley. I'm a data and software engineer who loves making stuff.

I've made software to automatically detect sequencing errors in NGS at Ultima Genomics, developed a software pipeline at Stanford which makes finding single cell DNA mutations in cancer possible, assisted the development of a neural network model for classifying cell types to characterize SARS-CoV2 at Berkeley BAIR. Currently I am working on an open source HPC regression testing platform for large linux server clusters with Dow Inc. for my Master's Capstone Project. I also like designing and making stuff, of which you can see examples below.

Cell Designer

A web application to design and render 3D models of cells and solve how they interact. Still a WIP. You can drag the 3d model around to get a better view of the cell model!

gpt-flow, an llm flowchart

A flow chart large language model runner, fully deployed over aws with frontend written in typescript/React and a backend written in python using Flask.

veccy, a RAG application

A RAG application that improves semantic search with query prefetching and implements a built in web crawler that preprocesses scraped webpages with an LLM. Written in React js with backend in python FastAPI, pinecone for vector database.
